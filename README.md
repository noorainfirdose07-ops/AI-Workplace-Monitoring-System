**🖥️ AI Workplace Monitoring System**
An intelligent AI-powered system for real-time workplace activity monitoring, productivity analysis, and employee behavior tracking.

**📌 Project Overview**
  The AI Workplace Monitoring System is a real-time employee activity monitoring solution that combines Artificial Intelligence, Computer Vision, and Cloud Technologies to enhance workplace productivity.
  By leveraging YOLOv8 object detection, OpenCV image processing, Flask backend services, and Supabase cloud database, the system intelligently detects workplace-related objects 
      (person, laptop, mobile phone) and classifies employee activities into categories such as Working, Idle, Distracted, and Away.
 The system continuously processes live webcam feeds, generates activity logs with timestamps, and stores monitoring records securely in the cloud.
  A user-friendly dashboard provides live monitoring, analytics, and reporting, enabling organizations to track productivity, identify distractions, and improve workflow management.
  Designed with scalability in mind, the architecture supports future enhancements such as face recognition, attendance tracking, multi-user monitoring, AI alerts, and advanced analytics dashboards.
  This project demonstrates the practical application of AI and Deep Learning in workplace automation, offering a secure, efficient, and intelligent solution for modern organizations.

**▶️ How to Use**
  Clone the repository and install dependencies (pip install -r requirements.txt).
  Configure Supabase credentials in .env.
  Run the Flask application (python app.py).
  Access the monitoring dashboard at http://127.0.0.1:5000/.
  View live detection outputs, activity logs, and productivity reports.
